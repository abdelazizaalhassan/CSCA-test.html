What Is a Set?

A set is a well-defined collection of distinct objects called elements. Sets use curly braces and can be described by listing or by a rule.

B = {6, 7, 8, 9, 10}
Set-Builder Form

Quadratic Inequalities

The 4-Step Method

Algorithm тАФ Never Miss a Step Step 1: Rearrange so one side is 0: ax┬▓ + bx + c тЧЛ 0
Step 2: Find the roots by factoring or quadratic formula
Step 3: Apply the parabola rule (a > 0 assumed for standard form):

Step 4: Choose bracket type: тЙе or тЙд тЖТ include roots [ ]; strict > or < тЖТ exclude roots ( )
Visual Memory тАФ The Upward Parabola Rule An upward parabola (a > 0) is above zero outside the roots and below zero between them.
Think of it as a "U" shape: the bottom dips below the x-axis (negative zone) sits between the roots.

Rational Inequalities

Why You Cannot Cross-Multiply

A rational inequality has the form f(x)/g(x) тЧЛ 0. You cannot multiply both sides by g(x) to clear the denominator тАФ because g(x) might be negative, which would reverse the inequality direction.

The Sign Chart Method тАФ 5 Steps 1. Move everything to one side тЖТ f(x)/g(x) тЧЛ 0
2. Find critical values: where numerator = 0 тЖТ candidate solutions; where denominator = 0 тЖТ always excluded
3. Mark all critical values on a number line, dividing it into intervals
4. Test a point in each interval; determine sign (+/тИТ) of the whole expression
5. Select intervals matching the inequality; include numerator zeros if тЙд or тЙе; never include denominator zeros
Iron Rule тАФ Denominator Zeros Are Always Excluded The denominator can never be zero тАФ the function is undefined there. Even if the inequality is тЙд or тЙе, a point where the denominator is zero is excluded from the solution, always.

тЬУ Unit 1 Complete Summary тАФ Sets & Inequalities
  • тИИ connects elements to sets; тКЖ connects sets to sets. Never swap them.
  • The empty set ╬ж is a subset (тКЖ) of every set тАФ not necessarily an element.
  • Intersection A тИй B: take the larger left endpoint & smaller right endpoint; use the more restrictive bracket.
